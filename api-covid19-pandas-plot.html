Exportar datos¶

Finalmente, se guardaron los datos en CVS, el formato más común de importación y exportación de hojas de cálculo y bases de datos, a tráves del código vs.to_csv('vs.csv'). A su vez, estos datos también pueden exportarse como archivo PNG, así como se hizo en la actividad con la función: grafico = vs.plot() fig = grafico.get_figure() fig.savefig("vs.png") Este último formato queda guardado en la carpeta que se ha creado para el proyecto.

In [68]:
vs.to_csv('vs.csv')
